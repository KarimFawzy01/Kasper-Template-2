# 🎨 Kasper-Template-2 | Elzero HTML & CSS Template

![Kasper Preview](images/screenshot.png)

## 📝 عن المشروع
**Kasper-Template-2** هو ثاني قالب من 3 قوالب تطبيقية قمت بتنفيذها أثناء دراسة كورس **HTML & CSS** مع المهندس أسامة الزيرو على يوتيوب. القالب عبارة عن موقع شركة/وكالة خدمات، وتم تصميمه بالكامل باستخدام HTML و CSS فقط.

## ✨ المميزات
- ✅ تصميم عصري وجذاب.
- ✅ متجاوب بالكامل مع جميع الأجهزة (Responsive).
- ✅ استخدام **Flexbox** و **CSS Grid** في توزيع العناصر.
- ✅ أقسام واضحة: Home, Services, Portfolio, About, Pricing, Contact.
- ✅ فيديو خلفية يظهر كخلفية لعنصر/مقطع معين في التصميم.
- ✅ أيقونات من **Font Awesome** وخطوط من **Google Fonts**.
- ✅ كود نظيف ومنظم (Semantic HTML).

## 🛠️ التقنيات المستخدمة
- **HTML5** (Semantic Elements)
- **CSS3** (Flexbox, CSS Grid, Custom Properties, Media Queries)
- **Video Background** (فيديو كخلفية لعنصر معين)
